# 📊 Business Sales Performance Analytics

## 📌 Project Overview

This project focuses on analyzing an Online Retail Sales dataset to evaluate business performance and identify valuable sales insights. The data was cleaned and processed using Python in Google Colab, followed by the creation of an interactive dashboard in Power BI to visualize key business metrics and trends.

---

## 🎯 Project Objectives

- Analyze monthly revenue trends
- Measure total sales and quantity sold
- Analyze customer purchasing activity
- Compare revenue with unit price
- Build a professional business dashboard for decision-making

---

## 🛠️ Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Power BI
- CSV Dataset

---

## 🧹 Data Cleaning & Preprocessing

The dataset was cleaned using Python in Google Colab.

### Steps Performed

- Imported the dataset
- Checked dataset information
- Removed duplicate records
- Handled missing values
- Converted data types
- Created a new column:

```python
TotalSales = Quantity × UnitPrice
```

- Exported the cleaned dataset as CSV
